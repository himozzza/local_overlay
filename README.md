# polkit-pipeline
Автоматическое обновление polkit*.ebuild в локальном оверлее в соответствии с доступной версией исходников на freedesktop, для получения более свежей версии polkit-duktape.

Зависимости:
1. python-3.9
2. eix
3. wget
